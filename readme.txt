1.“智能众包管理平台”已部署在云服务器上，访问地址 http://118.25.40.254/tp5/public/index.php 为登录界面，建议使用火狐浏览器，可以自己注册，在注册过程中需填写手机号，所在公司，若提示此公司不存在，请填写“吃瓜小队”，需要录入人脸信息，点击拍照上传4张人脸信息，然后点击注册，注册后需要管理员审核才能登入系统。

2.软件包在解压后需要修改数据库连接和文件地址，把tp5/application/api/model下的所有数据库地址密码修改为自己的，在tp5/application/api/controller下的文件中把文件的地址修改为该文件所在地址，然后导入数据库就可使用。

3.将tp5文件解压后按照上述方法修改后，访问自己服务器目录中的tp5/public/index.php就可进入登录界面。
